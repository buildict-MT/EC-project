# 企画書：製造部品特化型 EC サイト「RecruitDirect」

| 項目 | 内容 |
|------|------|
| **プロジェクト名** | RecruitDirect (リクルートダイレクト) |
| **対象商材** | コンベアローラ、搬送ベルト、ベアリングユニット、アルミフレーム |
| **作成日** | 2026-04-07 |

---

## 1. 背景・目的
現在、製造業界における部品選定と発注業務は、依然としてアナログなプロセス（電話、FAX、仕様の書面やり取り）が多く、以下の課題がある。

- **選定の手間**: 数値の微差による仕様の不一致が発生しやすい。
- **価格の不透明性**: 毎回見積り依頼を出す必要があり、即時判断ができない。
- **リードタイムの長期化**: 発注前の「選定〜見積り」の往復に時間がかかる。

本プロジェクトの目的は、**「Web上での即時仕様選定・即時価格算出」**を実現し、設計者・購買担当者の業務スピードを飛躍的に向上させることである。

## 2. ターゲット
- **装置設計者**: 径・長さ・材質などを選定し、CADデータをダウンロードしたいユーザー。
- **購買担当者**: 見積書を即時発行し、稟議をスムーズに進めたいユーザー。
- **現場保全担当**: 故障した部品を型番やカテゴリから迅速に再注文したいユーザー。

## 3. 解決策 (Value Proposition)
- **選定シミュレーター**: パラメータ変更に連動して形状（SVG）と価格がリアルタイムに変化。
- **即時見積発行**: カートに入れた選定品をそのまま PDF 見積書として出力可能。
- **B2B 実用性**: CAD データ、仕様書の即時提供。

## 4. 期待される効果
1.  **リードタイム削減**: 見積り待ち時間をゼロにする。
2.  **発注ミス低減**: 視覚的なプレビューと自動計算により、不整合な仕様の注文を未然に防ぐ。
3.  **利便性向上**: いつでもどこでもタブレットからでも選定・発注が可能になる。
