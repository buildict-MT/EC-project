# 実装報告書：RecruitDirect プロジェクト

## 1. 概要
本報告書は、製造部品特化型 B2B EC サイト「RecruitDirect」における一連の実装および設計変更作業を記録したものである。旧名称「MakerDirect」からのリブランド、および B2B 実運用に耐えうる高度な機能設計と実装を完了した。

## 2. 実施内容

### 2.1 プロジェクト・リブランディング
- **プロジェクト名変更**: 全ソースコード、ドキュメント（README、設計書）、UI（ヘッダー、フッター、ブラウザタイトル）における「MakerDirect」を「RecruitDirect」へ統一。
- **リポジトリ管理**: GitHub リポジトリ (`https://github.com/buildict-MT/EC-project`) への初期登録および継続的なプッシュを完了。

### 2.2 設計の拡張（B2B 特化要件）
既存の設計書を大幅に拡張し、以下の定義を追加・統合した：
- **組織・権限設計**: 企業・部署単位の管理、および4つのロール（管理者/承認者/発注担当/設計担当）の定義。
- **承認ワークフロー**: 発注担当者による「承認申請」から承認者による「確定」までのフロー。
- **価格戦略**: 数量に応じたボリュームディスカウント（10個以上で5%引、50個以上で10%引）のロジック。
- **システム連携**: 基幹システム（ERP）連携および CAD データ動的生成のアーキテクチャ設計。

### 2.3 バックエンド実装 (Node.js / Express / SQLite)
- **データベース構築**: SQLite を使用し、B2B に必要な 6 つの主要テーブル（companies, users, products, specs, orders, order_items）を構築。
- **認証システム**: `jsonwebtoken` (JWT) と `bcryptjs` (パスワードハッシュ化) を使用したセキュアなログイン API を実装。
- **注文 API**: ユーザーのロールに応じた注文ステータス管理（pending_approval / approved）および、トランザクションを考慮した注文登録 API を実装。
- **プロキシ設定**: フロントエンドからの API リクエストを適切に処理するため、`package.json` に `proxy` 設定を追加し、開発環境での CORS 等の問題を解消。

### 2.4 フロントエンド実装 (React / TypeScript)
- **認証状態管理**: `AuthContext` を新規作成し、アプリ全体でのログイン状態とユーザー情報の永続化を実現。
- **TypeScript 環境の最適化**: 
    - `Set` のスプレッド演算子展開における `TS2802` エラーを解消するため、`tsconfig.json` に `downlevelIteration: true` を追加。
- **ナビゲーションとレイアウトの強化**:
    - **自動スクロール**: ページ遷移時に最上部へスクロールする `ScrollToTop` 機能の実装。
    - **404 ページ**: 存在しないパスへのアクセスに対するエラーページの導入。
    - **全ページリンクの開通**: フッターおよび各ページ内のリンク（サポート、会社概要、採用情報等）をすべて有効化。
- **新規画面の追加と詳細化**:
    - **サポート関連**: 選定ガイド、CADデータ案内、納期確認、お問合せフォームを個別の詳細ページとして実装。
    - **会社情報関連**: 会社案内（ミッション・ビジョン・沿革・詳細な企業情報）、採用情報（カルチャー・募集職種・選考プロセス）を詳細に実装。
    - **採用エントリーフロー**: 求人へのエントリーフォーム、およびカジュアル面談リクエストフォームを新規実装。
- **PDF 見積書生成の安定化**:
    - 日本語フォント (Noto Sans JP) の埋め込み。
    - `fontkit` インスタンスの登録により、カスタムフォント使用時のランタイムエラーを解消。
- **管理機能の拡充**:
    - 管理コンソールにおけるステータス別フィルタリングの実装。
    - 注文一覧からの詳細ステータスの直接更新機能の実装。
- **システム連携**:
    - **CAD データ連携**: 選定パラメータに基づいた DXF ファイルの自動生成とダウンロード機能を実装。

## 3. 使用技術スタック
- **Frontend**: React (Hooks, Context API), React Router, Lucide-React, pdf-lib, @pdf-lib/fontkit
- **Backend**: Express.js, SQLite3, jsonwebtoken, bcryptjs, cors
- **Tools**: Git, npm, TypeScript, Esbuild
