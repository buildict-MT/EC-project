## 1. 機能要件

### 1.1 商品一覧・検索機能
- カテゴリ（ローラ、ベルト等）ごとの商品一覧表示。
- 商品名、価格、特徴のサマリー表示。
- 詳細画面への遷移リンク。

### 1.2 商品詳細・選定シミュレーター機能
- **パラメータ選択**: 径 (φ)、材質 (ST/SUS/樹脂)、長さ (L) などの動的選択。
- **価格計算ロジック**: ベース価格 ＋ 仕様加算 ＋ 寸法加算（100mm 単位）を即時反映。
- **SVG プレビュー**: 選択した径・長さ・材質に基づいた、簡易的な形状イラストの動的描画。
- **関連データ提供**: CADデータ（DXF等）や仕様書（PDF等）のダウンロードリンク。

### 1.4 会員・権限管理機能 (B2B特化)
- **組織管理**: 企業、部署、ユーザーの3階層管理。
- **ロールベースアクセス制御 (RBAC)**: 管理者（全権限）、承認者（注文承認）、発注担当（カート投入・申請）、設計担当（選定のみ）の権限分離。
- **承認ワークフロー**: 発注担当による「注文申請」から承認者による「承認」を経て正式受注となるフロー。

### 1.5 注文・配送・決済管理機能
- **B2B決済**: クレジットカード、掛売（請求書払い）、銀行振込への対応。
- **注文ステータス追跡**: 受注、製造中、出荷済、配送中、完了の各ステータス管理。
- **再注文機能**: 過去の注文履歴（選定仕様含む）からのワンクリック再注文。

### 1.6 商品・在庫・価格戦略機能
- **ボリュームディスカウント**: 同一商品の注文数量に応じた段階的な単価割引。
- **動的納期回答**: 工場負荷と材質・仕様に基づく標準出荷日のリアルタイム算出。
- **在庫引当**: 部材（パイプ、軸受等）の在庫状況に基づく製作可否判定。

### 1.7 API・システム連携機能
- **外部連携API**: 基幹システム（ERP）や生産管理システムとのデータ同期。
- **CADデータ連携**: 選定パラメータに基づいたSTEP/DXFデータの動的生成または紐付け。

## 2. 非機能要件

### 2.1 パフォーマンス・信頼性
- 価格計算およびプレビュー更新の応答速度（0.5秒以内）。
- PDF 生成およびダウンロードの待機時間（2秒以内）。
- ブラウザ上での動作（Chrome, Edge 最新版推奨）。
- **可用性**: 稼働率 99.9% 以上の維持。

### 2.2 デザイン・ユーザビリティ
- 工業的な美しさと信頼感を感じさせる配色、フォント（メイキブルー）。
- レスポンシブ対応（PC、タブレット、スマートフォン）。
- 複雑な選定でも迷わない直感的な UI。

### 2.3 セキュリティ・運用
- **データ保護**: 通信の常時 SSL/TLS 化、機密情報の暗号化保存。
- **脆弱性対策**: SQL インジェクション、XSS、CSRF 等への対策。
- **ログ管理**: 操作ログ、アクセスログの 1 年間保存。


## 3. データ要件

### 3.1 商品マスタ
- 商品ID、カテゴリ、ベース名、ベース価格、説明文。

### 3.2 仕様マスタ
- 商品IDに関連する仕様名、表示値、加算価格。
