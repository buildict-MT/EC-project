## 1. 機能要件

### 1.1 商品一覧・検索機能
- カテゴリ（ローラ、ベルト等）ごとの商品一覧表示。
- 商品名、価格、特徴のサマリー表示。
- 詳細画面への遷移リンク。

### 1.2 商品詳細・選定シミュレーター機能
- **パラメータ選択**: 径 (φ)、材質 (ST/SUS/樹脂)、長さ (L) などの動的選択。
- **価格計算ロジック**: ベース価格 ＋ 仕様加算 ＋ 寸法加算（100mm 単位）を即時反映。
- **SVG プレビュー**: 選択した径・長さ・材質に基づいた、簡易的な形状イラストの動的描画。
- **関連データ提供**: CADデータ（DXF等）や仕様書（PDF等）のダウンロードリンク。

### 1.3 カート・見積機能
- 選定した仕様を含む商品のカート追加、数量変更、削除。
- カート内の小計、送料、合計金額のリアルタイム算出。
- **見積書 PDF 発行**: カート内容（仕様含む）を整理した見積書をブラウザ上で生成・ダウンロード。

## 2. 非機能要件

### 2.1 パフォーマンス・信頼性
- 価格計算およびプレビュー更新の応答速度（0.5秒以内）。
- PDF 生成およびダウンロードの待機時間（2秒以内）。
- ブラウザ上での動作（Chrome, Edge 最新版推奨）。

### 2.2 デザイン・ユーザビリティ
- 工業的な美しさと信頼感を感じさせる配色、フォント（メイキブルー）。
- レスポンシブ対応（PC、タブレット、スマートフォン）。
- 複雑な選定でも迷わない直感的な UI。

## 3. データ要件

### 3.1 商品マスタ
- 商品ID、カテゴリ、ベース名、ベース価格、説明文。

### 3.2 仕様マスタ
- 商品IDに関連する仕様名、表示値、加算価格。
