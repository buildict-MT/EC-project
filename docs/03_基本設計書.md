## 1. 画面設計・機能網羅

### 1.1 ホーム画面 (`/`)
- **ブランド訴求**: メインコピーおよび「選ばれる理由」セクション。
- **カテゴリ誘導 (REQ-1.1)**: ローラ、ベルト、ベアリングへの直接リンク。
- **新製品紹介**: 動的な商品カード表示。

### 1.2 商品一覧画面 (`/products`) (REQ-1.1)
- **カテゴリフィルタ**: サイドバーまたはヘッダーでのカテゴリ絞り込み機能。
- **商品サマリー**: 商品名、説明、最小参考価格を表示。
- **詳細リンク**: 各カードからシミュレーター画面へ遷移。

### 1.3 商品詳細・選定シミュレーター (`/products/:id`) (REQ-1.2)
- **パラメータ選定**:
  - **ローラ径 (φ)**: セレクトボックスによる選択。
  - **材質**: ボタン選択（ST / SUS 等）。
  - **ローラ幅 (L)**: 100mm〜2000mm の範囲を 10mm ステップのスライダーで選択。
- **動的技術プレビュー (REQ-1.2)**: 
  - 選択した仕様を反映した SVG イラストを中央に配置。
  - 数値表示（φXX × LXXXX）を併記。
- **関連ドキュメント (REQ-1.2)**:
  - 「CADデータ (DXF)」ダウンロードボタン（ダミー）。
  - 「仕様書 (PDF)」ダウンロードボタン（ダミー）。
- **リアルタイム価格表示**: 仕様変更ごとに税込価格を更新。

### 1.5 ログイン・企業認証画面 (`/login`)
- **認証フォーム**: 企業ID、ユーザーID、パスワードの入力。
- **組織選択**: 複数組織に所属する場合の企業切替機能。

### 1.6 マイページ・注文履歴 (`/account`)
- **会員プロフィール**: 自身の権限（設計・発注・承認）と所属情報の表示。
- **注文履歴一覧**: 過去の注文内容の照会、PDF 見積書・領収書の再発行。
- **リピート注文**: 履歴から同一仕様の商品をカートへ再投入。

### 1.7 管理画面 (`/admin`)
- **ユーザー管理**: 自社メンバーの追加・削除、権限（ロール）の変更。
- **承認待ちリスト**: 承認者のみアクセス可能な未承認注文の承認・却下機能。
- **価格・納期設定**: 特定顧客向けの特別割引率の設定（オプション）。

---

## 2. 帳票仕様 (見積書・請求書 PDF)

- **見積書**: 有効期限の明記、特記事項の動的挿入。
- **請求書**: 締日指定に基づく月次請求書（掛売用）のフォーマット対応。

---

## 3. システム連携アーキテクチャ

- **フロントエンド ↔ バックエンド**: REST API (JSON) を使用。
- **ERP 連携**: バッチ処理またはリアルタイム Webhook による在庫・納期情報の同期。
- **CAD 生成エンジン**: 選定結果を受け取り、サーバサイドで 3D/2D データを生成し、URL をフロントに返却。

---

## 4. 非機能要件の実現方針

- **セキュリティ (NF-2.3)**: 
  - HTTPS 強制。
  - セッションタイムアウト（30分無操作で自動ログアウト）。
  - パスワードのハッシュ化（Argon2 等）保存。
- **可用性 (NF-2.1)**: 
  - クラウド基盤のマネージドサービス活用による高可用性構成。
