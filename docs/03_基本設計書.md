## 2. 画面設計・機能網羅

### 2.1 ホーム画面 (`/`)
- **ブランド訴求**: 製造現場の選定を加速させるブランドメッセージ。
- **機能概要**: 即時シミュレーション、B2B承認ワークフロー、CAD提供の訴求。

### 2.2 商品一覧画面 (`/products`) (REQ-1.1)
- **検索・フィルタ**: キーワード検索およびカテゴリ別の絞り込み機能。
- **商品サマリー**: 商品名、説明、最小参考価格を表示。

### 2.3 商品詳細・選定シミュレーター (`/products/:id`) (REQ-1.2)
- **パラメータ選定**: ローラ径、材質、ローラ幅 (L) の動的選択。
- **動的技術プレビュー**: 仕様を反映した SVG イラストのリアルタイム更新。
- **価格計算**: 仕様加算、長さ加算、およびボリュームディスカウントの適用。

### 2.4 ログイン・企業認証画面 (`/login`) (REQ-1.4)
- **認証機能**: 企業ID/ユーザーIDによるログイン、JWTの発行。

### 2.5 マイページ・注文履歴 (`/account`)
- **履歴管理**: 自身の過去の注文・申請内容の確認、ステータス照会。

### 2.6 管理画面 (`/admin`) (REQ-1.5)
- **承認フロー**: 承認待ち注文の一覧表示、および承認・却下アクション。

---

## 3. 帳票仕様 (見積書・請求書 PDF)

- **見積書**: 企業名（宛名）の動的反映、仕様明細の整理出力。
- **請求書**: 月次請求（掛売）対応用フォーマット（将来拡張）。

---

## 4. システム連携・非機能要件

- **通信**: フロントエンド ↔ バックエンド間は REST API (JSON) を使用。
- **セキュリティ**: HTTPS、パスワードのハッシュ化、JWTベースの認可制御。
- **レスポンシブ**: PC、タブレット、モバイル（480px〜）の3段階対応。
