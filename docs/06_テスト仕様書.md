## 2. 選定・価格計算エンジンテスト (REQ-1.2, 1.6 / DD-1.1)

|----|-----------|---------|---------------|---------------|------|----------|
| 2.1 | 仕様加算 (Spec) | DD-1.1 | 径 `φ50` (+500) を選択 | 単価が +500円加算されていること | | |
| 2.2 | 長さ加算 (L) | DD-1.1 | L=100 (+500) と L=190 を比較 | 両者とも `Math.floor(L/100)` により同一価格 (+500) であること | | |
