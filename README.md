# RecruitDirect - 導入・起動マニュアル

RecruitDirect（リクルートダイレクト）は、製造部品（ローラ、ベルト等）に特化した B2B 対応型 EC サイトのプロトタイプです。

---

## 1. 動作環境
- **Node.js**: v18 以降推奨
- **npm**: v9 以降
- **OS**: Windows / macOS / Linux

---

## 2. セットアップ手順

### 2.1 依存関係のインストール
プロジェクトのルートディレクトリで以下のコマンドを実行します。

```bash
npm install --legacy-peer-deps
```

### 2.2 開発サーバーの起動
Frontend (React) と Backend (Express) を同時に起動します。

```bash
npm run dev
```

- **Frontend**: `http://localhost:3000`
- **Backend (API)**: `http://localhost:5000`

---

## 3. 主要機能の使い方

### 3.1 商品選定シミュレーター
1. [商品一覧] から製品を選択。
2. 詳細画面で「ローラ径」「材質」「長さ」を調整。
3. **動的プレビュー**: 仕様に合わせて SVG が変化。
4. **価格計算**: 仕様に応じた金額が自動算出。

### 3.2 見積書 (PDF) の発行
1. 商品を [カートに入れる] で追加。
2. カート画面へ移動。
3. [見積書を発行 (PDF)] をクリック。

---

## 4. データベースについて
開発用として **SQLite3**（ファイルまたはメモリ）を使用しています。
- **将来的な計画**: 本番環境への移行時には、スケーラビリティとデータ整合性確保のため **PostgreSQL** への変更を予定しています。

---

## 5. 開発者向け情報

### 5.1 B2B 特化機能
- **認証**: JWT を用いたセキュアなログイン。
- **組織管理**: ユーザーごとのロール（admin, approver, orderer）による権限制御。
- **承認フロー**: カート投入後の承認申請および、管理者による承認・却下プロセス。

### 5.2 自動テスト
以下のコマンドで API の自動テストを実行できます。
```bash
npm test
```
※ Vitest および Supertest を使用しています。
